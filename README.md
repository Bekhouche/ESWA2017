# ESWA2017 : Pyramid Multi-Level Features for Facial Demographic Estimation
a draft code for :
- Demographic estimation [PAL]
- Age estimation [FG-NET]
- Gender Classification [FG-NET and PAL]
- Ethnicity classification [PAL]
